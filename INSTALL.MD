# Basic Install Guide
1. Install ox_lib if it isn't already installed https://github.com/CommunityOx/ox_lib
2. Install motion_bridge if it isn't already installed https://github.com/Motion-Scripts/motion_bridge
3. Download latest release.
4. Add script to your Fivem resources folder.
5. Add `ensure motion-redzones` to `server.cfg`
6. Start to config the script
- Add any zones you need in shared/config.lua
- Setup webhooks in shared/server_config.lua
- Setup kill tracking settings in shared/server_config.lua

# Dependencies
motion_bridge (for ambulance and inventory)
ox_lib
